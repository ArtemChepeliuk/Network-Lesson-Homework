1. Alt-Svc: hq=":443"; ma=2592000; quic=51303431; quic=51303339; quic=51303335,quic=":443"; ma=2592000; v="41,39,35" -----
Це означає, що браузер, якщо захоче, може використовувати QUIC , це HTTP над UDP, через порт 443 наступні 30 днів (ma = 2592000 секунд, або 720 годин, тобто 30 днів)
2. Content-Length: 0 	------ Розмір вмісту контенту в октетах (які зазвичай називають байтами).
3. Content-Type: text/html ----------- тип контенту - text/html
4. Date: Wed, 21 Mar 2018 12:36:48 GMT ---------- дата
5. Location: https://www.youtube.com/ ---------URI за яким клієнт має перейти або URI створеного ресурсу.
6. Server: YouTube Frontend Proxy ------------ Список назв і версій веб-сервера і його компонентів з коментарями
7. Strict-Transport-Security: max-age=31536000; includeSubDomains; preload --------- HTTP Strict Transport Security (HSTS) це механізм політики безпеки, який дозволяє захистити сайт від спроби небезпечного з'єднання. max-age=31536000 час, в секундах, яке браузер повинен пам'ятати про заголовку. includeSubDomains Якщо вказати це опциональное значення, то заголовок поширюється і на всі піддомени. preload якщо власник сайту хоче, щоб домен потрапив в зумовлений список, який підтримується Chrome
8. X-Xss-Protection: 1; mode=block ------------- Атака XSS (міжсайтовий скриптинг) це тип атаки, при якому шкідливий код може бути впроваджений в атакуєму сторінку. 1; mode = block. Фільтр включений, але якщо атака виявиться, сторінка не буде завантажена браузером.
